# Xin chào! Tôi là Tài

Tôi là một lập trình viên đam mê công nghệ, đặc biệt yêu thích lập trình mạng và phát triển web.

## Kỹ năng

### Ngôn ngữ lập trình
- **Java**: Có kinh nghiệm với Java Core, Java 8+, Spring Framework
- **JavaScript**: Thành thạo ES6+, Node.js, và các framework hiện đại
- **Python**: Sử dụng cho automation và data processing

### Công nghệ
- Backend: Spring Boot, Node.js, Express
- Frontend: React, Vue.js, HTML/CSS
- Database: MySQL, PostgreSQL, MongoDB
- Tools: Git, Docker, Maven
