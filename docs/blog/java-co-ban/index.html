public class MayTinh {
    public static void main(String[] args) {
        int a = 10, b = 5;
        String phepTinh = "+";
        
        int kq = switch (phepTinh) {
            case "+" -> a + b;
            case "-" -> a - b;
            case "*" -> a * b;
            case "/" -> (b != 0) ? a / b : 0;
            default -> 0;
        };
        
        System.out.println(a + " " + phepTinh + " " + b + " = " + kq);
    }
}

Lời khuyên

  • Đặt tên biến/phương thức: camelCase (vd: tenNguoiDung)
